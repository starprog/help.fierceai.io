# My Documents

## Overview

**My Documents** is your personal library for organizing and managing all AI-generated content you've saved. It provides a clean, folder-based system to keep your creative work organized and easily accessible.

Think of it as your curated collection - only the content you explicitly save appears here, keeping your workspace focused and clutter-free.

## Troubleshooting

### Documents Not Appearing

**Problem**: Saved content doesn't show in My Documents

**Solutions**:
1. **Check Content Manager**: Document might not be saved to My Documents
2. **Filter Settings**: Ensure "All" filter is selected
3. **Folder View**: Check if document is in a folder
4. **Refresh Page**: Browser cache might be outdated
5. **Storage Limit**: You may have reached storage capacity

### Cannot Delete Document

**Problem**: Delete button not working or error appears

**Solutions**:
1. **Check Permissions**: Ensure you own the document
2. **Active Generation**: Wait for generation to complete
3. **Team Access**: Admin role might be required
4. **Browser Issue**: Try different browser or clear cache

### Folder Not Loading

**Problem**: Clicking folder doesn't show contents

**Solutions**:
1. **Empty Folder**: Folder might have no documents
2. **Network Issue**: Check internet connection
3. **Permissions**: Verify folder access rights
4. **Corrupted Data**: Contact support if persists

### Missing Content

**Problem**: Document content is blank or missing

**Solutions**:
1. **Generation Failed**: Check if generation completed successfully
2. **Storage Issue**: File might have been deleted from storage
3. **Cloud Sync**: Cloud storage might be temporarily unavailable
4. **Restore from Backup**: Contact support for recovery options
