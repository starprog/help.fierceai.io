## ‚úèÔ∏è Managing Contacts

### Viewing Contact Details

Each contact entry shows you:
- Basic information (name, email, phone)
- Which channel they used
- When they first contacted you
- Last interaction timestamp

### Editing Contact Information

1. Click the **pencil icon** (‚úèÔ∏è) next to any contact
2. Update the following fields as needed:
   - **Name**: Full name or display name
   - **Email**: Email address for follow-up
   - **Phone**: Phone number for direct contact
3. Click **"Save"** to update the record

**Why Edit Contacts?**
- Correct misspelled names or incorrect information
- Add missing email or phone details learned from other interactions
- Standardize naming conventions across your contact list
- Merge duplicate information from multiple conversations

### Deleting Contacts

1. Click the **X icon** next to the contact you want to remove
2. Confirm the deletion when prompted
3. ‚ö†Ô∏è **Warning**: This action is permanent and cannot be undone

**When to Delete:**
- Test conversations during chatbot setup
- Spam or bot interactions
- Duplicate contacts
- Upon user request (GDPR/privacy compliance)

## üí° Best Practices

### Contact Data Quality

**Encourage Information Sharing**
- Design chatbot flows that naturally request contact details
- Explain benefits of providing email (e.g., "Get instant email confirmations")
- Offer incentives (discounts, early access) for complete profiles

**Verify Critical Information**
- Use email validation during collection
- Format phone numbers consistently
- Implement data quality checks

**Keep Records Clean**
- Regular review and cleanup of duplicate entries
- Merge similar contacts when identified
- Remove test data and spam entries

### Privacy & Compliance

**Respect User Privacy**
- Only collect information necessary for your use case
- Clearly communicate how data will be used
- Honor deletion requests promptly

**GDPR Compliance**
- Obtain explicit consent before collecting personal data
- Provide easy access to view/download personal information
- Delete data upon request
- Maintain records of consent

**Data Security**
- Contact data is encrypted and securely stored
- Access limited to authorized users only
- Regular security audits and updates

## üö® Troubleshooting

### Common Issues

**Contacts Not Appearing**
- **Check Chatbot Status**: Ensure chatbot is published and active
- **Verify Embed Code**: Confirm chatbot is properly embedded on your site
- **Test Interaction**: Send a test message to create a contact record
- **Clear Cache**: Browser cache may prevent real-time updates

**Duplicate Contacts**
- **Same User, Multiple Sessions**: Users starting new conversations create new records
- **Different Channels**: Same person on website vs social media may appear as separate contacts
- **Manual Cleanup**: Identify duplicates by email/phone and delete extras

**Missing Contact Information**
- **User Didn't Provide**: Not all users share personal details
- **Chatbot Configuration**: Ensure chatbot asks for contact information
- **Form Design**: Make information requests clear and user-friendly
- **Edit Manually**: Add information learned from other sources

**Cannot Edit/Delete Contact**
- **Permission Issues**: Verify you have account access rights
- **Page Refresh**: Try refreshing the browser
- **Browser Compatibility**: Use modern browser (Chrome, Firefox, Safari, Edge)
- **Contact Support**: If issues persist, reach out to support@fierceai.io
