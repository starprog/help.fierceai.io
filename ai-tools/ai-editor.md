### How Tokens Are Consumed

AI Editor uses token-based billing for content generation and AI-powered editing features:

| Action | Token Usage | Description |
|--------|-------------|-------------|
| **Generate Content** | ~1 token per word | Using templates to create new content |
| **AI Editing** | ~1 token per word edited | AI-powered refinements and suggestions |
| **Content Expansion** | ~1 token per word | Extending existing content |
| **Tone Adjustment** | ~1 token per word | Changing tone or style with AI |
| **Manual Editing** | Free | Regular text editing without AI |
| **Organizing/Saving** | Free | File management operations |

## FAQ

**Q: How many workbooks can I create?**  
A: Unlimited workbooks. Organize as many projects as you need.

**Q: Can I share workbooks with team members?**  
A: Yes, workbooks can be shared with team members on Team plans.

**Q: Are my documents automatically saved?**  
A: Yes, AI Editor auto-saves your work every few seconds.

**Q: Can I import existing content?**  
A: Yes, you can copy/paste or import from various file formats.

**Q: How long are documents stored?**  
A: Documents are stored indefinitely until you delete them.

**Q: Can I edit on mobile devices?**  
A: Yes, AI Editor is responsive and works on tablets and phones.

**Q: Is there a document size limit?**  
A: Individual documents can be up to 50,000 words.

**Q: Can I restore deleted content?**  
A: Deleted content can be restored from trash within 30 days.
