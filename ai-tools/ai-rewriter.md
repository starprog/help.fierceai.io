### How Tokens Are Consumed

AI Rewriter uses token-based billing calculated on the output (rewritten) text:

| Content Length | Approximate Tokens | Typical Usage |
|----------------|-------------------|---------------|
| **Short Paragraph** (100 words) | ~100 tokens | Quick rewrites, social posts |
| **Medium Content** (500 words) | ~500 tokens | Blog paragraphs, emails |
| **Long Article** (1,500 words) | ~1,500 tokens | Full articles, reports |
| **Extended Content** (3,000+ words) | ~3,000+ tokens | White papers, ebooks |

### Token Calculation

- **Input doesn't count**: Only the rewritten output uses tokens
- **1 token ‚âà 1 word** in most languages
- **Shorter modes use fewer tokens**: "Shorten" mode produces less output
- **Expansion uses more**: "Expand" mode generates additional content

### Token-Saving Tips

1. **Choose the Right Mode**: Use "Shorten" to reduce token usage
2. **Process in Sections**: Rewrite long content in chunks
3. **Be Specific**: Clear mode selection prevents re-rewrites
4. **Review Before Rewriting**: Fix obvious errors manually first
5. **Use Simple Mode**: Basic rewrites use fewer tokens than creative expansions

**For Professional Polish:**

**For Creativity:**
- **Creative** - Marketing copy, engaging content

## Rewriting Modes Explained

### Professional Mode

**Best For**: Business communications, corporate content, professional emails

**Transformations:**
- Removes slang and casual language
- Adds formal tone and structure
- Improves business vocabulary
- Enhances professional presence

**Example:**
```
Input: "Hey, just wanted to let you know the report's gonna be a bit late. 
Been super busy with other stuff."

Output: "I am writing to inform you that the report will be delivered 
slightly behind schedule due to competing priorities requiring immediate attention."
```

### Casual Mode

**Best For**: Social media, friendly emails, blog posts, newsletters

**Transformations:**
- Relaxes formal language
- Adds conversational elements
- Makes content approachable
- Improves readability

**Example:**
```
Input: "It is imperative that all personnel submit their documentation prior 
to the established deadline."

Output: "Hey everyone! Just a heads up - make sure to get your docs in before 
the deadline. Thanks!"
```

### Creative Mode

**Best For**: Marketing copy, engaging content, storytelling, ads

**Transformations:**
- Adds personality and flair
- Uses vivid language
- Incorporates metaphors
- Enhances engagement

**Example:**
```
Input: "Our product is good and affordable."

Output: "Discover a solution that doesn't just meet expectations‚Äîit shatters 
them. Quality excellence meets wallet-friendly pricing in perfect harmony."
```

### Academic Mode

**Best For**: Research papers, scholarly articles, thesis writing

**Transformations:**
- Adds academic rigor
- Uses scholarly vocabulary
- Improves citation-readiness
- Enhances formal structure

**Example:**
```
Input: "Studies show that people work better when happy."

Output: "Empirical research demonstrates a significant positive correlation 
between employee satisfaction levels and productivity metrics across multiple 
organizational contexts (Smith et al., 2023)."
```

### Simplify Mode

**Best For**: Technical documentation, educational content, explanations

**Transformations:**
- Reduces complexity
- Breaks down jargon
- Shortens sentences
- Improves accessibility

**Example:**
```
Input: "The implementation of synergistic methodologies facilitates optimal 
resource allocation paradigms."

Output: "Working together helps us use our resources more effectively."
```

### Expand Mode

**Best For**: Adding detail, meeting word counts, comprehensive explanations

**Transformations:**
- Adds supporting details
- Includes examples
- Expands on concepts
- Increases depth

**Example:**
```
Input: "Exercise is good for health."

Output: "Regular physical exercise provides numerous health benefits, including 
cardiovascular improvements, enhanced mental well-being, increased muscle strength, 
better sleep quality, and reduced risk of chronic diseases such as diabetes and 
hypertension. Studies consistently demonstrate that even moderate exercise, such as 
30 minutes of daily walking, can significantly improve overall health outcomes."
```

### Shorten Mode

**Best For**: Summaries, social media posts, quick versions

**Transformations:**
- Removes redundancy
- Keeps key points
- Reduces word count
- Maintains essentials

**Example:**
```
Input: "In order to successfully complete this project, it is absolutely essential 
and critically important that all team members work together collaboratively and 
communicate effectively with one another on a regular basis."

Output: "Project success requires team collaboration and regular communication."
```

### Improve Mode

**Best For**: General content enhancement, quality improvement

**Transformations:**
- Fixes grammar and spelling
- Improves sentence flow
- Enhances word choice
- Polishes overall quality

**Example:**
```
Input: "Their going to the store to buy some stuff for there project tomorrow 
and it's gonna be great."

Output: "They're going to the store tomorrow to purchase materials for their 
project, which will be excellent."
```

## Troubleshooting

### Rewrite Doesn't Match Expectations

**Solution:**
- Try a different rewriting mode
- Add specific custom instructions
- Break content into smaller sections
- Provide more context in the input

### Output Too Similar to Original

**Solution:**
- Choose a stronger transformation mode (Creative, Professional)
- Use custom instructions for specific changes
- Try running a second rewrite with different mode
- Ensure sufficient input length for meaningful transformation

### Token Usage Higher Than Expected

**Solution:**
- Use Shorten mode to reduce output length
- Process content in smaller chunks
- Choose simpler modes (Improve vs. Creative)
- Review input length - longer input = longer output

### Language Output Incorrect

**Solution:**
- Verify correct language selected from dropdown
- Ensure input is in a supported language
- Try rewriting again if language mixed
- Contact support if language not working

### Content Loses Important Details

**Solution:**
- Add specific instructions to preserve key elements
- Use Improve mode instead of aggressive transformations
- Break into sections and rewrite only parts that need it
- Review output immediately and note missing elements

## FAQ

**Q: Can I rewrite content in multiple languages?**  
A: Yes, AI Rewriter supports 40+ languages. Select your target language before rewriting.

**Q: How long can my input content be?**  
A: You can rewrite content up to 5,000 words in a single request. For longer content, split into sections.

**Q: Will rewriting make my content unique?**  
A: Yes, AI Rewriter creates original rewrites. However, always review for quality and verify facts.

**Q: Can I rewrite the same content multiple times?**  
A: Yes, you can rewrite as many times as needed. Each rewrite will use tokens.

**Q: Does AI Rewriter check for plagiarism?**  
A: AI Rewriter creates original content, but we recommend using plagiarism checkers for important publications.

**Q: Can I save different versions of rewrites?**  
A: Yes, save each rewrite to your workbook to compare versions and track changes.

**Q: What's the difference between Improve and Professional modes?**  
A: Improve enhances overall quality while keeping the original tone. Professional specifically converts to business-appropriate tone.

**Q: Can I undo a rewrite?**  
A: The original content remains in the input area. Simply regenerate if needed or use version history in your workbook.

**Q: How accurate is the rewriting?**  
A: AI Rewriter preserves meaning and context accurately. Always review output, especially for technical or critical content.

**Q: Can I rewrite technical or specialized content?**  
A: Yes, but provide sufficient context. AI handles technical content well when given appropriate domain information.
