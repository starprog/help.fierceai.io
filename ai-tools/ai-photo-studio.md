## Troubleshooting

### Background Removal Issues

**Problem:** Subject edges are rough or incomplete

**Solutions:**
- Ensure good contrast between subject and background
- Use higher resolution source image
- Try with clearer lighting
- Check that subject is in focus

### Upscaling Not Meeting Expectations

**Problem:** Upscaled image looks artificial or blurry

**Solutions:**
- Start with highest quality source possible
- Don't upscale beyond 4x original size
- Try processing in smaller increments
- Ensure source isn't heavily compressed

### Text to Image Results Don't Match Vision

**Problem:** Generated image doesn't match description

**Solutions:**
- Be more specific in descriptions
- Include style keywords (realistic, artistic, minimalist)
- Mention lighting and mood
- Try multiple variations
- Add technical details (angle, composition)

### Sketch to Image Quality

**Problem:** Result doesn't capture sketch intent

**Solutions:**
- Use clearer, more defined lines
- Add text description for style guidance
- Ensure sketch has good contrast
- Try with simpler sketches first

## FAQ

**Q: Can I process the same image with multiple tools?**  
A: Yes! Download the result after each tool and use it as input for the next.

**Q: What's the maximum image size I can upload?**  
A: Recommended maximum is 10MB. Larger files may be automatically compressed.

**Q: How long does processing take?**  
A: Most operations complete in 5-15 seconds depending on complexity and image size.

**Q: Can I download original source images?**  
A: Yes, originals are never modified and can be re-downloaded from your history.

**Q: What image formats are supported?**  
A: JPG, PNG, and WEBP are supported. PNG recommended for transparency workflows.

**Q: Can I undo or regenerate?**  
A: You can regenerate using the same settings. Original files are always preserved.

**Q: How many images can I process?**  
A: No limit on number of images, only token balance limits individual operations.

**Q: Is there a quality difference between ClipDrop and Novita?**  
A: Both are professional quality. Your admin can set the default based on specific needs.

**Q: Can I use generated images commercially?**  
A: Yes, generated images are yours to use. Check terms for specific third-party integrations.

**Q: What happens to failed generations?**  
A: Failed attempts don't consume tokens. You'll be notified and can try again.
