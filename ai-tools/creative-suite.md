### How Tokens Work

Creative Suite uses **two token types** depending on the feature:

**IMAGE Tokens** - Used for:
- AI image generation from text prompts
- Image variations and enhancements  
- AI-powered photo effects

**WORD Tokens** - Used for:
- AI text actions (longer, shorter, better)
- Content generation and improvement
- Text enhancements within the editor

### Token Calculation

**For Images:**
```
Tokens Used = Number of Images √ó Token Rate
```

**For Text:**
```
Tokens Used = Number of Words Generated √ó Token Rate
```

### Example Calculations

**Image Generation:**
- Generating 2 images at 1.0 rate: **2 IMAGE tokens**
- Generating 2 images at 1.5 rate: **3 IMAGE tokens**
- Generating 4 images at 0.5 rate: **2 IMAGE tokens**

**Text Enhancement:**
- Improving 50-word text at 1.0 rate: **~50 WORD tokens**
- Making content longer (50 ‚Üí 100 words) at 1.0 rate: **~100 WORD tokens**
- Making content shorter (100 ‚Üí 50 words) at 1.0 rate: **~50 WORD tokens**

## Getting Started

### Step 1: Access Creative Suite

1. Log in to your FierceAI dashboard
2. Navigate to **Creative Suite** from the main menu
3. You'll see your recent documents and templates

### Step 2: Start a New Project

**From Scratch:**
1. Click **"Create New"**
2. Choose your canvas size
3. Start designing

**From Template:**
1. Browse template library
2. Select a template that matches your needs
3. Customize to your requirements

### Step 3: Generate AI Images

1. Click the **AI Generate** button in the toolbar
2. Select your preferred AI model
3. Enter a detailed text description
4. Set image parameters:
   - Number of images (1-4)
   - Image style and mood
   - Quality settings
5. Click **"Generate"**
6. Wait for AI to create your images
7. Images appear on your canvas

### Step 4: Edit and Enhance

**Layer Tools:**
- Add text layers
- Insert shapes and elements
- Arrange layer order
- Group related layers

**Editing Tools:**
- Crop and resize
- Adjust colors and brightness
- Apply filters and effects
- Remove backgrounds

**Text Tools:**
- Choose from hundreds of fonts
- Adjust size, color, and spacing
- Add text effects and shadows
- Create text on paths

### Step 5: Save and Export

1. Click **"Save"** to store your project
2. Name your document
3. Choose export format:
   - PNG (transparent backgrounds)
   - JPG (photos and graphics)
   - PDF (print-ready)
   - SVG (scalable vectors)
4. Download to your device

## Pro Tips

### Image Generation Tips

1. **Be Descriptive**: Detailed prompts produce better results
2. **Specify Style**: Mention desired art style (watercolor, photorealistic, etc.)
3. **Include Details**: Colors, lighting, composition, mood
4. **Use Examples**: Reference famous artists or styles
5. **Iterate**: Generate multiple versions and refine

### Design Best Practices

1. **Use Layers**: Organize elements on separate layers
2. **Brand Consistency**: Maintain consistent colors and fonts
3. **White Space**: Don't overcrowd your design
4. **Hierarchy**: Guide the viewer's eye with size and placement
5. **Export Settings**: Choose the right format for your use case

### Workflow Optimization

1. **Save Often**: Protect your work with frequent saves
2. **Template Library**: Create reusable templates
3. **Batch Generate**: Generate multiple options at once
4. **Organize Projects**: Use clear, descriptive names
5. **Monitor Tokens**: Track usage to manage costs

### Canvas Not Loading

**Possible Causes:**
- Browser compatibility issues
- Cache or cookies
- Network connection

**Solutions:**
- Use modern browsers (Chrome, Firefox, Edge)
- Clear browser cache
- Check internet connection
- Disable browser extensions

### Export Issues

**Possible Causes:**
- Large file size
- Format incompatibility
- Storage space

**Solutions:**
- Reduce image resolution
- Try different export format
- Check available storage
- Contact support if persists
