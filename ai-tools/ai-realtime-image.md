## Writing Effective Prompts

### Real-Time Prompt Strategy

Since AI Realtime Image updates as you type, structure your prompts strategically:

1. **Start with the Subject**
   ```
   A red apple
   ```

2. **Add Key Details**
   ```
   A red apple on a wooden table
   ```

3. **Refine with Modifiers**
   ```
   A shiny red apple on a rustic wooden table, soft lighting
   ```

4. **Polish with Final Touches**
   ```
   A shiny red apple on a rustic wooden table, soft natural lighting, warm tones
   ```

### Prompt Templates

**Product Visualization**
```
[Product] on [surface], [lighting style], [mood], professional photography
```
Example:
```
Modern headphones on white surface, studio lighting, minimalist, professional photography
```

**Character Design**
```
[Character description], [style], [pose/action], [setting], [mood]
```
Example:
```
Friendly robot character, cartoon style, waving gesture, futuristic city, cheerful
```

**Scene Creation**
```
[Location/Setting], [time of day], [weather/atmosphere], [style], [mood]
```
Example:
```
Mountain landscape, golden hour sunset, misty atmosphere, realistic, serene
```

**Abstract Concepts**
```
[Concept] represented as [visual metaphor], [style], [color palette]
```
Example:
```
Innovation represented as glowing network of connections, modern, blue and purple
```

## Frequently Asked Questions

**Q: How fast is "real-time"?**
A: Images typically generate in 1-3 seconds, providing near-instant feedback as you type.

**Q: Can I generate multiple images at once?**
A: No, AI Realtime Image focuses on single-image real-time generation. For batch generation, use AI Image Generator.

**Q: What resolution are the images?**
A: Images are generated at a fixed resolution optimized for speed and quality balance.

**Q: Can I use generated images commercially?**
A: Check your FierceAI plan terms. Most plans allow commercial use of generated images.

**Q: How is this different from AI Image Generator?**
A: AI Realtime Image prioritizes speed for rapid iteration, while AI Image Generator offers multiple models, higher quality options, and variable resolutions.

**Q: Can I edit the generated images?**
A: Yes! Export to AI Image Editor for advanced editing capabilities.

**Q: Is there a limit to how many images I can generate?**
A: Limited by your token balance. Each generation consumes IMAGE tokens based on your plan.

**Q: Why do some prompts work better than others?**
A: Flux Schnell understands natural language well, but clear, descriptive prompts with specific details tend to produce better results.

**Q: Can I regenerate a previous image?**
A: Yes, click on any saved image in your gallery to see its prompt and regenerate with modifications.

**Q: What happens if generation fails?**
A: Tokens are not deducted for failed generations. Check your connection and try again.
